Experiência do usuario e User Interface

O UX Design,é o profissional que foca na experiência do usuário, como o próprio termo UX (user experience) sugere. Ele se preocupa com a forma que cada componente afeta o usuário. Por ser mais direcionado a sensações, é mais subjetivo, ligado às impressões de alguém sobre o uso de uma interface. Por isso, essa área está associada com a Psicologia, sendo importante buscar alguns conceitos e uma certa visão de mundo desse campo. O profissional de UX precisa de bastante empatia, pois seu trabalho envolve a vivência e as impressões que outras pessoas terão. Assim, a área estuda a percepção de valor de alguém ao usar uma interface. Está preocupada com a eficiência, com facilidade, praticidade e com os problemas do usuário. Ela se pergunta se o produto realmente ajudou o cliente em suas necessidades e se cooperou para que se sentisse satisfeito.


UI Design

O termo UI significa “user interface” (interface do usuário) e refere-se aos componentes visuais que fazem parte de toda aplicação, permitindo ou não a interação de pessoas, tais como: os botões, as caixas de texto, a tipografia, as cores e imagens de uma página. A composição visual dos sistemas serve para guiar o usuário dentro dos objetivos de quem programou, com o mesmo propósito de um manual. Em outras palavras, se tomarmos como exemplo o aplicativo de alguma rede social, veremos que ele foi desenhado a fim de permitir que as pessoas publiquem pensamentos e ideias, compartilhem links, adicionem amigos, vejam posts em um feed, entre outras ações.

    A sigla UI significa User Interface, em português, Interface do Usuário. Ela representa tudo aquilo que é utilizado na interação com um produto. Sendo a intermediária visual (ou não visual) entre o homem e a máquina (celular, computador, TV).
